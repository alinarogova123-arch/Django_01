# Пульт охраны банка
Показывает сколько всего пропусков есть с допуском к хранилищу, а так же сколько из них активны.

## Как установить

Вам потребуется адрес и пароль для базы данных, а так же secret key сайта. 

1. `DB_HOST` - Адрес базы данных.
2. `DB_PASSWORD` - Пароль доступа пользователя к базе данных.
3. `SECRET_KEY` - Секретный ключ сайта.

Адрес базы данных присвоить переменной окружения `DB_HOST`, пароль присвоить переменной окружения `DB_PASSWORD`,
`SECRET_KEY` сайта присвоить переменной окружения `DB_SECRET_KEY` в файле .env как в примере ниже:

``` bash
DB_HOST=...
DB_PASSWORD=...
DB_SECRET_KEY=...
```

Скрипт запускается командой:
```
python main.py
```

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```
